# Taloyhtiön kirjanpitosovellus

Tämä sovellus on harrastemielessä toteutettu työkalu taloyhtiön kirjanpidon helpottamiseen. Sovelluksen avulla voidaan ylläpitää taloyhtiön tili- ja tapahtumatietoja, tehdä talousarvioita sekä seurata tilinpäätöstä ja vuosittaista kehitystä. Sovellus tukee kaksipuolista kirjanpitoa ja mahdollistaa myös esimerkkidatan käytön.

Sovellus on toteutettu kokonaan "vibe-koodaten" ja omaksi iloksi ja oppimismielessä ilman kaupallisia tavoitteita.

## Ominaisuudet

- Taloyhtiön perustietojen hallinta
- Tilien ja tapahtumien ylläpito
- Kaksipuolinen kirjanpito
- Talousarvion laadinta ja seuranta
- Poistojen ja jaksotusten kirjaaminen
- Tilinpäätöksen tulostus
- Esimerkkidatan käyttö ja testaus

## Huomio

Sovellus on kehitetty harrasteprojektina, eikä sitä ole tarkoitettu tuotantokäyttöön ilman jatkokehitystä ja testausta. Kaikki palaute ja kehitysehdotukset ovat tervetulleita!
